## How To Read This

- `mixedRejects` means VP did try shared candidates, but the usable set was blocked by one or more real rejection causes. These are the best follow-up targets.
- `onlyDuplicate` means a valid shared design existed, but it was rejected as a duplicate of the already represented design. These are mostly Mk II / variant opportunities, not emergency missing coverage.
- `targetTonsMax` is the practical design ceiling to aim for. It already reflects shipyard and tech-tonnage constraints where the log had them.
- `techTonnageTons=?` means the log did not have a separate tech-tonnage value for that hull type or country in that entry.
